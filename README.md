# git-forensic

**Audit AI-authored code quality in git repositories.**

In 2026, ~4% of public GitHub commits are AI-authored — heading to 20% by year-end. `git-forensic` answers: **"How good is the AI code in my repo?"**

![Dashboard](https://raw.githubusercontent.com/hyunseung1119/forensic/main/report-screenshot.png)

## Features

- **AI Commit Detection** — identifies `Co-Authored-By: Claude/Copilot/GPT/Cursor/Gemini`, `Generated by` tags, and heuristic patterns
- **4-Dimension Quality Scoring** — Commit Message (25%), Change Size (30%), Test Coverage (30%), Documentation (15%)
- **Rich Terminal Output** — colored tables, progress bars, grade badges
- **HTML Dashboard** — single-file dark-theme report with donut charts, KPI cards, and commit details
- **JSON Export** — for CI/CD integration and custom analysis

## Install

```bash
# With uv (recommended)
uv tool install git-forensic

# Or pip
pip install git-forensic
```

## Usage

```bash
# Scan current repo
git-forensic .

# Scan with HTML dashboard (auto-opens in browser)
git-forensic /path/to/repo --html report.html --open

# Filter by date
git-forensic . --since 2026-01

# Export JSON for CI
git-forensic . -o report.json

# Show all commits (including low-confidence heuristic matches)
git-forensic . --all-commits
```

## Quality Dimensions

| Dimension | Weight | What it measures |
|-----------|--------|-----------------|
| Commit Message | 25% | Conventional commit format, length, descriptiveness |
| Change Size | 30% | Focused changes vs. massive rewrites |
| Test Coverage | 30% | Whether tests accompany the code change |
| Documentation | 15% | Docs commits, self-documenting messages |

## Detection Signals

| Signal | Confidence | Example |
|--------|-----------|---------|
| `Co-Authored-By: Claude` | 95% | Claude Code commits |
| `Co-Authored-By: GitHub Copilot` | 95% | Copilot suggestions |
| `Generated by Claude/GPT` | 85% | Explicit AI tags |
| `aider:` prefix | 90% | Aider CLI commits |
| Conventional commit + structured desc | <40% | Heuristic only |
